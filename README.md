# ping-chart
**Simple** chart to view network stability

# Install
Download the lastest version from [releases](https://github.com/nullxx/ping-chart/releases)
# Building
* Clone this repo
```bash
git clone https://github.com/nullxx/ping-chart
```
* Move to the folder
```bash
cd ping-chart
```

* Install dependencies
```bash
npm install
```
or
```bash
yarn
```
* Run
```bash
npm start
```
